All notable changes to this project are documented here. The format is based on

## [0.1.0]

### Added

- Initial release: DNI and NIE (`nie` module) validation — `is_valid`, `validate`,
  `calc_check_digit`, and `compact` — a zero-dependency, `no_std` port of the `python-stdnum`
  Spanish identification number algorithms.
